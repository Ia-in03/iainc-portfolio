# iainc-portfolio Webpage
This is the source code behind my portfolio website

## Getting Started

First, run the development server:

```bash
# Run locally in development mode
npm run dev

# Build and run in production mode
npm run build
npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel

GitHub actions will automatically deploy on Vercel when new changes are pushed/merged into main
View deployment dashboards [here](https://vercel.com/ia-in03s-projects/iainc-portfolio)
